# Архитектура проекта ESP32 AI Translator

## Основные блоки

- `main/` — приложение ESP-IDF для ESP32-S3.
  - `main.c` — главный цикл, инициализация аудио и переводчика.
  - `audio.c` / `audio.h` — абстракция захвата аудио с микрофона.
  - `translator.c` / `translator.h` — абстракция распознавания и перевода.

- `components/ai_translator/` — дополнительный компонент ESP-IDF.
  - `ai_translator.c` — шаблон для логики AI-компонента.

- `python/` — утилиты на Python для разработки и отладки.
  - `audio_client.py` — пример захвата аудио и интерфейса для распознавания.
  - `run_translator.py` — простой CLI-скрипт для тестовой записи и перевода.

- `scripts/` — вспомогательные скрипты.
  - `setup_env.sh` — настройка Python-окружения и зависимостей.
  - `build.sh` — сборка проекта ESP-IDF.
  - `flash.sh` — прошивка и монитор.

## Поток данных

1. Захват голоса с микрофона на ESP32-S3.
2. Обработка аудио-кадров и передача их в модуль распознавания.
3. Преобразование распознанного испанского текста в русский.
4. Вывод результата в журнал или дальнейшая передача в синтез речи.

## Расширение проекта

- Реализовать `audio_capture()` для реального микрофона.
- Подключить модель распознавания речи в `translator_recognize_speech()`.
- Добавить перевод с помощью локальной модели или облачного API в `translator_translate()`.
- Добавить синтез речи или отображение результата на экране.
