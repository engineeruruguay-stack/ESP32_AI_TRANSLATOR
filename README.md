# ESP32 AI Translator

Проект: голосовой переводчик на ESP32-S3 с аудиоплатой.

## Цели
- Переводить испанскую речь на русский.
- Работать в реальном времени.
- По возможности использовать Python и ESP-IDF.

## Структура проекта
- `main/` — основной код ESP-IDF.
- `components/ai_translator/` — вспомогательный компонент для AI-процессов.
- `python/` — утилиты и тестовые скрипты на Python.
- `scripts/` — вспомогательные сценарии для сборки и прошивки.
- `docs/` — документация и архитектурные заметки.

## Быстрый старт
1. Установите ESP-IDF и активируйте окружение по инструкции Espressif.
2. Запустите `scripts/setup_env.sh`.
3. Соберите проект: `scripts/build.sh`.
4. Прошивайте плату: `scripts/flash.sh /dev/ttyUSB0`.

## Python тесты
Используйте `python/run_translator.py` как шаблон для локального захвата и отладки аудио.

> В этом репозитории содержится скелет проектной структуры. Все блоки реализованы как каркас, который можно заполнять под конкретный аудио-поток и модель перевода.
